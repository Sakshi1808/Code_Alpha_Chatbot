# Code_Alpha_Chatbot
Simple ChatBot using Python
# Project Description
This is a basic rule-based chatbot created using Python.
The chatbot responds to simple user inputs like greetings, asking its name, and goodbye messages.
#The project demonstrates:
Python functions
Conditional statements (if-elif-else)
Loops
User input handling
String matching
# Features
Greets the user
Responds to basic questions
Runs continuously until the user types "bye"
Handles unknown inputs with a default response
